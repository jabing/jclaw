## 更新步骤

### 在 GitHub 上更新:

1. **打开仓库**: https://github.com/jabing/jclaw
2. **点击右上角 "⚙️ Settings"**
3. **在 "About" 区域:**
   - 粘贴英文简介 (选项 1 或 2 或 3)
   - 保存
4. **点击 "Manage topics"**
   - 删除所有旧 topics
   - 添加新的 25 个 topics (上面的列表)
   - 保存

### 或者使用 GitHub CLI:

```bash
# 更新简介
gh repo edit jclaw --description "Universal self-evolving Agent framework with persistent memory, AutoSkill, and skill.sh integration. Zero-dep, cross-platform."

# Topics 需要在 GitHub UI 中手动添加
# 访问：https://github.com/jabing/jclaw 点击 "Manage topics"
```
