## Contributing Translations

We welcome contributions for translations! If you'd like to help translate JClaw documentation into your language:

1. Create a new file: `README_[lang].md` (e.g., `README_ja.md` for Japanese)
2. Translate the content from `README.md`
3. Update this file with the new translation link
4. Submit a pull request

### Translation Guidelines

- Keep code examples in English (do not translate code)
- Translate comments in code examples
- Maintain the same structure and formatting
- Update version numbers and dates if needed
