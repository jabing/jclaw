## ❗ 常见错误排查

### 错误 1: 包含逗号
```
❌ ai-agents, self-evolving, persistent-memory
✅ ai-agents self-evolving persistent-memory
```

### 错误 2: 包含大写字母
```
❌ TypeScript NodeJS
✅ typescript nodejs
```

### 错误 3: 包含特殊字符
```
❌ ai/memory c++
✅ ai-memory cpp
```

### 错误 4: 包含空格在 topic 内部
```
❌ ai agents self evolving
✅ ai-agents self-evolving
```

### 错误 5: 以连字符开头或结尾
```
❌ -ai-agents ai-agents-
✅ ai-agents
```
