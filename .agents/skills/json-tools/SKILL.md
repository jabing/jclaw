# JSON Tools Skill

JSON parsing, validation, and transformation utilities.

## Capabilities

- **json_parse**: Parse JSON strings safely
- **json_validate**: Validate JSON against schema
- **json_transform**: Transform JSON structures
- **json_format**: Format/beautify JSON

## When to Use

Use this skill when:
- User mentions "JSON", "parse JSON", "validate JSON"
- Need to transform data formats
- Working with API responses
